The Mitigation Controller wraps everything. It short-circuits blocked IPs before wasting detection cycles, logs every flagged event to SQLite, manages timed blocks with auto-expiry, and escalates repeat offenders.
Decision tree — pick a request type to walk through the pipeline
🛑
Step 1 — Is this IP currently blocked?
Checks the SQLite ip_reputation table for an active block with unexpired unblock_at timestamp.
📋
Step 2 — Run Rule Engine (Phase 1)
Check auth, injection, rate, scan, payload in priority order. Short-circuit on first match.
📊
Step 3 — Run Anomaly Engine (Phase 2)
Only reached if all rule checks pass. Score IAT against per-IP baseline using modified Z-score.
⚖
Step 4 — Execute verdict
Act on the combined result. Log to SQLite. Increment counters.

Escalation: deprioritise strikes → auto-block

A single anomalous request is deprioritised (forwarded but tagged). Repeated anomalies from the same IP indicate sustained suspicious activity. The escalation system tracks strikes in a sliding window — when the count reaches the threshold, the IP is automatically upgraded to a full timed block.

►
Strike threshold: 8 strikes
Why not 1? Why not 20?
Set to 8 to provide a small buffer against jitter false positives (occasional high-Z requests that aren't attacks) while still firing reliably when 50 concurrent burst requests arrive. A threshold of 1 would block on any anomalous request, causing false positives during normal traffic variation.
►
Window: 120 seconds
Old strikes expire and don't haunt the IP
A brief network anomaly generating 3 strikes shouldn't result in a block 10 minutes later. The 120s sliding window means only recent strikes count. Strikes older than 120s are evicted from the deque, giving the IP a clean slate over time.
►
Block duration: 30 seconds
Short for demo repeatability; production would be 5–15 min
30 seconds lets the demo be repeated without long waits. In a production system, escalation blocks would typically be 5–15 minutes. The duration is configurable via the /idms/config endpoint.
